Data Layer — On-Premise (126TB)

All data stored locally for zero-latency access. No cloud data egress costs.

Cryptocurrency Data

Kaiko Consolidated (11TB)

Tick-level order book snapshots (up to 5,000 bid/ask levels) and raw trades across 64 cryptocurrency exchanges. 10 years of history. Covers spot, perpetual futures, quarterly futures, and options markets.

Kaiko EQ Versions (v3/v4)

Enhanced quality datasets with standardized symbology, cross-exchange normalization, and derived metrics. Multiple versions for backward compatibility.

US Equity High-Frequency Data

NYSE TAQ (14TB)

Millisecond-resolution trades and quotes for all US-listed equities. Daily TAQ and intraday TAQ (DTAQ). The gold standard for equity microstructure research. NBBO, trades, quotes with exchange identifiers.

NASDAQ ITCH (12TB)

NASDAQ TotalView-ITCH feed data. Multiple versions (v2–v5) spanning full history. Order-level data: add, execute, cancel, replace messages. Enables order-flow and queue-position analysis.

WRDS Academic Databases

WRDS Cloud (272 databases)

Direct PostgreSQL connection to the full WRDS suite. Real-time queries against the latest data without local storage constraints.

WRDS On-Premise Mirror (12TB)

Local mirrors of high-frequency WRDS data. Order book snapshots at large/medium/small tick sizes (ob_l, ob_m, ob_s). Eliminates query latency for intensive computations.

Specialized Databases (6TB on-premise)

CRSP & Compustat

Complete US stock returns (CRSP) and financial statements (Compustat). Linked CRSP-Compustat merged dataset. The backbone of empirical finance.

IBES & Thomson Reuters

Analyst earnings estimates, recommendations, and target prices. Institutional holdings (S12/S34). Essential for event studies and analyst behavior research.

OptionMetrics

US equity options: implied volatility surfaces, Greeks, open interest, volume. Enables volatility research and options microstructure analysis.

TRACE (Fixed Income)

Corporate bond transaction data from FINRA. Trade prices, volumes, and dealer identifiers. The definitive source for bond market microstructure.

RavenPack (News Analytics)

Real-time news sentiment scores, event detection, and entity mapping. Covers 300,000+ entities across global news sources. Links news events to market movements.

Capital IQ & Factset

Company fundamentals, M&A transactions, private company data (CapitalIQ). Multi-asset class analytics, ownership data, supply chain mappings (Factset).

MSCI ESG & GMI Ratings

Environmental, social, and governance scores. Board composition, executive compensation, shareholder activism data. ESG integration research.

AI Model Layer

Claude Opus 4 — Orchestrator

Primary agent managing the full research lifecycle: data exploration → hypothesis → code → analysis → figures → paper → iteration. Stateful memory across sessions. Voice-controlled via Telegram.

Gemini 3.1 Pro — Deep Reasoning

Long-form academic writing, complex analytical reasoning, cross-model review. Publication-quality prose with proper econometric methodology discussions.

Gemini Flash — Multimodal

Audio transcription (10x Whisper accuracy for Chinese-English code-switching). Native audio generation for podcasts and video narration. Video analysis and rapid summarization.

Stateful Experience System

Structured error log (错题本) prevents repeated mistakes. Technical state persists across sessions; research judgments start fresh to avoid confirmation bias.

Design Philosophy

Stability Over Speed

Run at 80% capacity, never 100%. Prevent OOM crashes and data corruption. Every analysis script is independently runnable with built-in error handling.

Stateful for Technical, Stateless for Research

Technical details (file paths, field names, bugs) accumulate in a structured log. Research judgments start fresh — past conclusions don't constrain new hypotheses. Prevents confirmation bias while avoiding repeated mistakes.

Honest Replication

All discrepancies between original papers and our replications are documented transparently. When results differ from the literature, the difference is reported, not hidden.

Global Compute Leverage

Each API call rents access to infrastructure worth billions — Google's TPU clusters, Anthropic's GPU farms, exchange matching engines. The research budget buys a ~20,000,000× cost-to-infrastructure leverage ratio.
